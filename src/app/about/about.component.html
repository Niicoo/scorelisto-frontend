How it works

The conversion is divided in 3 main step:

- 1. Pitch Detection: By using a pitch detection algorithm [Mc Leod Pitch Detection Algorithm], we convert the input audio signal in a pitch array by applying the algorithm all along the track. Although the algorithm works with any sampling period, it is recommanded to record audio with a high sampling frequency such as 44.1Khz.

- 2. Step Detection: This stage consists in detecting the position of note changes.

- 3. Rhythm Detection: Now that the succession of the notes and their durations is known, we need to search the BPM and rhythms that fit the best for each notes.

Futurs improvements

- [Step Detection] Perform using machine learning.
- [Pitch Detection] Improve computation time.
